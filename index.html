Enhancement One: Software Design and Engineering

This enhancement focuses on improving the structure and organization of my SQL Injection Detection program from CS 405 Secure Coding. In the original version, the logic was less organized, which made the code harder to read and maintain.

For this enhancement, I refactored the program into smaller and more focused functions such as normalize_sql, has_where_clause, is_suspected_injection, execute_sql, and run_query. This made the code more readable, easier to maintain, and better structured overall.

This enhancement demonstrates my ability to apply software design principles by creating modular code that is easier to understand, reuse, and improve.
